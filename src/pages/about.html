ESP32 NAT Router

Extended Range & Guest Network Solution

ESP32 NAT Router is a simple microcontroller-based range extender that can also be used to create a second (guest) WiFi network.

This is a spare time project. If you have any questions or issues, feel free to ask at the GitHub project page.
